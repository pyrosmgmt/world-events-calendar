# Pyros World Events Calendar Research & Build Brief

> **Note:** This is a condensed version of the full specification.
> Expand sections as needed.

## Objective

Research, verify, and build a complete Google Calendar ecosystem for
Pyros using the best available public Google Calendar / iCalendar (.ics)
feeds. Where high-quality public feeds do not exist, generate custom
`.ics` calendars.

The final system should cover:

-   UK Sporting Events
-   US Sporting Events
-   Global Sporting Events
-   Combat Sports
-   Motorsport
-   Gaming
-   Entertainment & Pop Culture
-   Music & Festivals
-   Shopping Events
-   Holidays & Awareness
-   Technology & Social Media
-   Space & Science

## Requirements

1.  Research the best public calendar providers.
2.  Verify every provider is still active.
3.  Prefer official feeds where possible.
4.  Record pricing, update frequency, reliability and installation
    method.
5.  Recommend the best provider for each category.
6.  Generate fallback `.ics` calendars where no reliable feed exists.
7.  Produce installation guides, comparison tables and validation
    scripts.

    README.md

-   Provider comparison

-   Python generator
-   Python validator

## Success Criteria

-   Every category researched.
-   Every recommendation verified.
-   No fabricated URLs or providers.
-   Custom calendars generated where required.
-   Ready for Google Calendar import/subscription.
